# Crypto NFT

A modern cryptocurrency and NFT dashboard application built with React, featuring a rich UI with Material UI, Chakra UI, and Bootstrap components.

**Live URL:** [https://your-live-url.netlify.app/](https://ibneabbas-visionary-horse-b2cc1e.netlify.app/)

## 🚀 Tech Stack

- **React 18** - UI library
- **React Router DOM v6** - Routing and navigation
- **Material UI (MUI)** - Premium React component library
- **Chakra UI** - Accessible React component library
- **Bootstrap 5 & React-Bootstrap** - Responsive UI components
- **Styled Components & Emotion** - CSS-in-JS styling
- **Framer Motion** - Animation library
- **React Pro Sidebar** - Collapsible sidebar component
- **React Multi Carousel** - Carousel/slider component
- **React Scroll** - Smooth scrolling animations
- **AOS (Animate on Scroll)** - Scroll-based animations
- **React Icons** - Icon library
- **SASS** - CSS preprocessor

## 📋 Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
